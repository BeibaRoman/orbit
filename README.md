# Orbit — лендінг сторінка

Односторінковий лендінг для вигаданого SaaS-продукту. Навчальний проєкт на React, зроблений для практики компонентної архітектури, CSS-модулів та композиції компонентів.

## Демо

🔗 [Дивитись онлайн](https://beibaroman.github.io/orbit/)

![Orbit screenshot](./screenshot.png)

## Технології

- React 19
- Vite
- CSS Modules
- modern-normalize

## Запуск локально

\`\`\`bash
git clone https://github.com/BeibaRoman/orbit.git
cd orbit
npm install
npm run dev
\`\`\`

## Архітектурні рішення

- Перевикористовувані layout-компоненти (`Container`, `Section`, `SectionTitle`) замість дублювання стилів у кожній секції
- Дані відокремлені від розмітки (`data.js`), передаються через пропси з єдиної точки входу (`App.jsx`)
- CSS-модулі для ізоляції стилів кожного компонента
- Умовний рендер (наприклад, бейдж "Популярний" у блоці тарифів) та рендер колекцій через `.map()`
